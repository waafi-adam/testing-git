hello git hub

[git init] - intitialize as git repo
[git add .] - add files to staging area
[git commit -m "initial commit"] - commit

once you have initialised a file into git hub as a repostiatry you can make some changes and push it into git hub again as a different commit so it wont change affect you had made before, example as seen below:
git push .
git commit -m "background changed to red"
git push -u origin master
